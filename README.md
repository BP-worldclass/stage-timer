# Stage Timer — 설치 및 사용 가이드

---

## 1. Node.js 설치 (최초 1회)

1. https://nodejs.org 접속
2. **LTS 버전** 다운로드 (왼쪽 초록 버튼)
3. 설치 파일 실행 → 모두 Next → Install
4. 설치 완료 후 PC 재시작

---

## 2. 서버 실행

1. 이 폴더(stagetimer)를 PC 바탕화면 등 원하는 위치에 복사
2. `서버시작.bat` 더블클릭
3. 검은 창이 열리며 아래와 같이 표시되면 성공:

```
=================================
  Stage Timer 서버 시작됨
=================================

  📺 뷰어 (TV/미러링용)
     http://192.168.x.x:3000/viewer.html

  🎛  컨트롤러 (운영자용)
     http://192.168.x.x:3000/controller.html

  🔑 컨트롤러 비밀번호: 1234
=================================
```

---

## 3. 현장 사용법

### 준비
- 노트북(서버)과 모든 기기를 **같은 WiFi**에 연결

### TV / 미러링 화면 (뷰어)
- 아이패드 Safari → `http://192.168.x.x:3000/viewer.html`
- 화면을 탭하면 전체화면으로 전환
- HDMI로 TV에 연결하면 뷰어 화면이 출력됨

### 운영자 (컨트롤러)
- 본인 기기 Safari/Chrome → `http://192.168.x.x:3000/controller.html`
- 비밀번호 `1234` 입력 후 입장

---

## 4. 비밀번호 변경

`server.js` 파일을 메모장으로 열어 아래 줄 수정:

```
const CONTROLLER_PASSWORD = '1234';  ← 원하는 번호로 변경
```

`public/controller.html` 파일에서도 동일하게:
```
const CORRECT_PW = '1234';  ← 동일하게 변경
```

저장 후 서버 재시작.

---

## 5. 문제 해결

| 증상 | 해결 |
|------|------|
| 다른 기기에서 접속 안 됨 | 노트북 방화벽에서 포트 3000 허용 |
| IP 주소 모름 | cmd → `ipconfig` → IPv4 주소 확인 |
| 창이 바로 닫힘 | Node.js 재설치 |
